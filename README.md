# 小账本 📒

一款简洁优雅的个人记账 Web 应用，帮助你轻松记录每一笔收支。

## 🌐 在线演示

👉 **[点击这里在线体验](https://marvelous-naiad-ca0037.netlify.app/)**

> 💡 提示：数据存储在浏览器本地，所有操作都在您的设备上进行，安全可靠。

## ✨ 功能特点

- 📝 **快速记账** - 支持收入/支出记录，多种分类可选
- 📊 **数据统计** - 月度趋势图表，分类占比分析
- 💾 **本地存储** - 数据存储在浏览器本地，安全可靠
- 📱 **响应式设计** - 完美适配手机和桌面端
- 🎨 **精美界面** - 现代化 UI 设计，操作流畅

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在本地启动，终端会显示访问地址（通常是 `http://localhost:5173`）

### 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **React 18** - 用户界面库
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Recharts** - 图表库
- **Lucide React** - 图标库
- **date-fns** - 日期处理

## 📱 功能说明

### 账单记录
- 点击右下角 ➕ 按钮添加新账单
- 支持选择收入/支出类型
- 丰富的分类选择（餐饮、交通、购物等）
- 可添加备注和选择日期

### 统计分析
- 查看近 6 个月收支趋势
- 分类支出/收入占比统计
- 直观的图表展示

### 数据管理
- 导出数据为 JSON、CSV、TXT 多种格式
- 支持导入 JSON、CSV 格式数据恢复
- 可清空所有数据重新开始
